⚙️ Sheet Sync Setup

This app syncs new registrations from your Google Sheet. Since the sheet contains PII, a Google Apps Script proxy runs as you and returns only group names + emojis — no PII ever leaves the sheet.

Column X stores emojis. Column Y stores lineup position. Column Z stores group block name. Column AA stores coordinator notes. Run fillEmojis() once to auto-suggest emojis.

Step 1. Open your spreadsheet → Extensions → Apps Script. Replace all code with the current version from the project doc, then save and deploy.

Step 2. Make sure row 1 has these headers (exact column doesn't matter — found dynamically):

  • A cell containing emoji → column X
  • A cell containing position → column Y
  • A cell containing group block → column Z
  • A cell containing notes → column AA
